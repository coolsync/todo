## Use RGB values to Color Elements

The RGB value for black looks like this:

rgb(0, 0, 0)

The RGB value for white looks like this:

rgb(255, 255, 255)



## **Use RGB to Mix Colors**



| Color  |         RGB          |
| :----: | :------------------: |
|  Blue  |   `rgb(0, 0, 255)`   |
|  Red   |   `rgb(255, 0, 0)`   |
| Orchid | `rgb(218, 112, 214)` |
| Sienna |  `rgb(160, 82, 45)`  |
